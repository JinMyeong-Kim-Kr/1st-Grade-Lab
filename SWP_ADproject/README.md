# Codechecker 
이 프로젝트는 Pyqt5를 이용하여, 사용자의 파이썬 파일을 읽고 코드 규칙(Coding Convention)을 적용하여 코딩 규칙이 지켜지지 않은 부분을 수정해주는 프로그램입니다.

# 코딩 규칙
코드 규칙(Coding Convention)이란 같은 코드를 작성하고 유지 보수하는 개발자들 사이에 규정된, 코드가 지켜야 할 규약입니다.
